Port Mirroring Help

You can configure the Port Mirroring parameters on this page.

    DHCP Server - Enable or Disable the server. If you disable the Server, you must have another DHCP server within your network or else you must configure the IP address of the computer manually.
  • Port Mirroring - Enable or Disable the port mirroring. If you enable the port mirroring, you must set the mirrored port and mirroring port which must be different.
  • Mirrored Port - Set the port to monitor the other port.
  • Mirroring Port - Set the port which will be monitored.

Click the Save button to save your configuration.

Note: The mirrored port and mirroring port must be different.
