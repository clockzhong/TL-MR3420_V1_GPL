LAN Help

You can configure the IP parameters of LAN on this page.

  • IPv6 Address Assign Type You can select Route Advertisement or DHCP6s

    • While you select Route Advertisement,you must enter IPv6 Address Prefix and IPv6 Address Prefix Length.

    While you select DHCP6s,you must enter two more informations about Start IPv6 Address and Start IPv6 Address.

Click the Save button to save your settings.
