VLAN Tag Help

You can configure the VLAN VID parameters for different application on this page.

  • Port1 VID - You can set VLAN ID for port1, and port1 can be set as bridge with WAN.
  • Port2 VID - You can set VLAN ID for port2, and port2 can be set as bridge with WAN.
  • Port3~4 VID - You can set por3~4 VID for normal router.
  • WAN For TR069 VID - You can set this VID for TR069 connection.

Click the Save button to save your configuration.
