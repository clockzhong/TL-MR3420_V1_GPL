Print Server Help

You can configure print server on this page.

There are three state of the print server, as follows:

  1. Online- Indicates the print service has been turned on, and now no user in the enjoyment of printing services. You can click "Stop" button to stop printing service.
  2. Offline- Indicates the print service feature is disabled. You can click "Start" button to start the print service.
  3. Busy- Indicates the print service has been turned on, but at this moment there are users in the enjoyment of printing services.
