帮助－上网控制规则管理

路由器提供方便而又强大的上网控制功能，可以对内部主机的上网行为进行控制。在这里，您可以将您之前设置好的“主机列表”“访问目标”“日程计划”灵活地组合起来，达到限制内部主机访问网络的目的。
开启上网控制：打开/关闭上网控制功能。
规则描述：对规则的简单描述。
主机列表：您所选择的主机列表。
访问目标：您所选择的访问目标。
日程计划：您所选择的日程计划。
通过：是否允许通过。
状态：只有设为“启用”的时候本条目所设置的规则才能生效。
配置：可以对条目进行修改或者直接删除。
举例说明：如果您打算限制局域网中MAC地址为00-11-22-33-44-AA的计算机的上网行为，仅允许其在周六与周日的18点至20点访问www.google.com，禁止局域网中的其它主机访问网络。您需要进行如下的设置：
1、在主机列表中添加一台主机，其列表名为PC_1，MAC地址为00-11-22-33-44-AA。
2、在访问目标中添加一个列表，其列表名为Target_1，选择域名模式，填入www.google.com。
3、在日程计划中添加一个列表，其列表名为Sched_1，选择周六和周日，开始时间为1800，结束时间为2000。
4、回到本设置页面，请启用上网控制功能，选择缺省过滤规则为“凡是不符合已设上网控制规则的数据包，禁止通过本路由器”。
5、点击“增加单个条目”按钮，在添加条目的页面中进行如下设置：
 - 填入唯一的规则名，例如：Rule_1；
 - 选择主机列表PC_1；
 - 选择访问目标Target_1；
 - 选择日程计划Sched_1；
 - 选择“允许通过”；
 - 选择“生效”，点击“保存”按钮。
完成上述的设置之后，您将看到的规则是：
ID 规则描述 主机列表 访问目标 日程计划 通过 状态 配置
1 Rule_1 PC_1 Target_1 Sched_1 允许 生效 编辑 删除
就像上面提到的那样，在详细的参数设置界面中，您可以进行列表的组合。
规则描述：您需要给访问控制规则一个唯一的描述。
主机列表：您必须为规则指定一个主机列表。
访问目标：指定目标列表，缺省为任何目标。
日程计划：指定日程列表，缺省为任何时间。
通过：允许/禁止通过。
生效：使规则生效/失效。
