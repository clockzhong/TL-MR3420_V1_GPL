帮助－日程计划列表设置

路由器提供方便而又强大的上网控制功能，可以对内部主机的上网行为进行控制。在这里，您可以设置日程计划列表，在某些时间控制内部主机的上网行为。
日程描述：您需要给制订的日程一个唯一的描述。
星期：一周中的几天。
时间：一天中的某个时间段。
举例说明：如果您打算限制局域网中MAC地址为00-11-22-33-44-AA的计算机的上网行为，仅允许其在周六与周日的18点至20点访问www.google.com，这时您需要利用日程计划提供的功能进行如下的设置：
1、点击“增加单个条目”按钮。
2、填写一个唯一的日程名，例如：Sched_1。
3、选择星期六与星期天。
4、填写开始时间：1800，填写结束时间：2000，保存设置。
完成上述的设置之后，您将看到的日程计划是：
ID 日程描述 星期 时间 配置
1 Sched_1 周六 周日 1800-2000 编辑 删除
在详细的参数设置界面中，我们提供了丰富的参数供您设置，包括：
日程描述：您需要给制订的日程一个唯一的描述。
星期：您可以自己选择星期，也可以快速设置为“每天”。
时间：您可以自己指定时间段，也可以快速设置为“全天24小时”
开始时间：设置开始时间，格式为“HHMM”。
结束时间：设置结束时间，格式为“HHMM”。
