帮助－流量统计
数据包统计时间间隔：
该数值决定了统计当前流量的时间间隔。
IP地址：
被统计主机的IP地址。
总流量：
数据包数－ 路由器总的收、发数据的包的个数。
字节数  － 路由器总的收、发数据的字节数。
当前流量：
数据包数－ 路由器在用户指定的数据包统计时间间隔中收、发数据的包的个数。
字节数  － 路由器在用户指定的数据包统计时间间隔中收、发数据的字节数。
ICMP Tx －路由器在用户指定的数据包统计时间间隔中发送到广域网的ICMP包的个数。
UDP Tx  －路由器在用户指定的数据包统计时间间隔中发送到广域网的UDP包的个数。
TCP SYN Tx  －路由器在用户指定的数据包统计时间间隔中发送到广域网的TCP SYN 包的个数。
